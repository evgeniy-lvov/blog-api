    Posts

    Create post

    curl -X POST \
  http://localhost:3000/api/v1/posts \
  -H 'Content-Type: application/json' \
  -d '{
    "post": {
        "username": "test_user",
        "title": "test_title",
        "content": "test_content",
        "user_ip": "1.1.1.1"
    }
}'

    The above command returns JSON structured like this:

    {
    "data": {
        "id": "1",
        "type": "post",
        "attributes": {
            "title": "test_title",
            "content": "test_content",
            "user_id": 1,
            "rating": 0,
            "created_at": "2019-07-25T10:02:16.227Z",
            "updated_at": "2019-07-25T10:02:16.227Z"
        }
    }
}

    This endpoint creates post.

    HTTP Request

    POST http://localhost:3000/api/v1/posts

    Query Parameters

    Parameter Default Description
    post true Hash with post attributes
    post[username] true Post author username (create if not exist)
    post[title] true Post title
    post[content] true Post content
    post[user_ip] true Author IP-v4 address

    Get top N posts by rating

    curl -X GET \
  'http://localhost:3000/api/v1/posts/top?n=1'

    The above command returns JSON structured like this:

    {
    "data": [
        {
            "id": "1",
            "type": "post",
            "attributes": {
                "title": "test_title",
                "content": "test_content"
            }
        }
    ]
}

    This endpoint retrieves top rating posts.

    HTTP Request

    GET http://localhost:3000/api/v1/posts/top?n=<N>

    URL Parameters

    Parameter Description
    N Number of top rating posts

    Rates

    Rate post

    curl -X POST \
  http://localhost:3000/api/v1/rates/ \
  -H 'Content-Type: application/json' \
  -d '{
    "rate": {
        "post_id": "1",
        "value": "5"
    }
}'

    The above command returns JSON structured like this:

    {
    "data": {
        "id": "1",
        "type": "post",
        "attributes": {
            "rating": 5
        }
    }
}

    This endpoint rates post.

    HTTP Request

    POST http://localhost:3000/api/v1/rates

    Query Parameters

    Parameter Default Description
    rate true Hash with rate attributes
    rate[post_id] true Post id
    rate[value] true Rate value ( value in 1...5 range )

    User IP's

    Get single IP's

    curl -X GET \
  http://localhost:3000/api/v1/user_ips/single/

    The above command returns JSON structured like this:

    [
    {
        "ip": "127.0.0.1",
        "users": [
            "user_test1",
            "user_test2"
        ]
    }
]

    This endpoint retrieves single IP-adresses from which posted different users.

    HTTP Request

    GET http://localhost:3000/api/v1/user_ips/single/

    Errors

    The Blog API uses the following error codes:

    Request with out of range rate[value] attribute:

    curl -X POST \
  http://localhost:3000/api/v1/rates/ \
  -H 'Content-Type: application/json' \
  -d '{
    "rate": {
        "post_id": "1",
        "value": "6"
    }
}'

    The above command returns JSON with validation error structured like this:

    {
    "value": [
        "value must be integer in 1...5 range"
    ]
}
    Error Code Meaning
    400 Bad request.
    422 Validation error with errors output.
